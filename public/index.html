This is my rendition of the popular "Among Us" game. The Back End is written in Java and front end written in HTML using ThymeLeaf with added scripts for timers. GameNyte

This was a final project for CodeFellows Advanced Javascript Course. This application took 4 days to get to where it is, and utilizes socket.io and others to allow for mutiplayer ability. This project was created to by a "tabletop" board game simulator. It has the ability to upload images of board games from the user to AWS S3 which all users entering the room will be able to see and move their tokens around on the game.

About Me

After completing high school, I attended Triton College and St. Xavier University in Chicago Illinois.In order to pay off student loans I accrued in college, I decided to enlist in the U.S. Army to take advantage of the Student Loan Repayment program.
